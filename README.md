# Extension for Moodle+R

立命館大学に新しく導入されたLMS「Moodle+R」をより使いやすくするためのGoogle Chrome向け拡張機能です。

## 機能の紹介

*   **ベターレイアウト**: 時間割を見やすくし、画面を広く使えるようにします。曜日ヘッダーへの日付表示、時限セルへの授業時間の表示、曜日時限テキストのトリミングなどの調整が含まれます。さらにシラバス情報が取得されている場合は時間割に先生の名前も自動表示されます。
*   **教職員モード**: リボンに教職員向けのリンク（教職員ポータル、Responや打刻など）を直接アクセスできるように追加します。（非教職員の場合はStudent PortalやCampus Web等のリンクを追加）
*   **ホームスキップ**: 使わないトップページをスキップして、直接ダッシュボード（時間割ページ）に遷移します。
*   **現在の授業をハイライト**: 現在行われている授業のコマが時間割上でハイライト表示され、今受けている授業が一目でわかるようになります。また、今日の曜日の列全体もハイライトされます。
*   **シラバス情報の表示・連携**: Moodleの授業ページに「シラバス」ボタンを追加し、ワンクリックでシラバスに直接アクセス。さらに、一度シラバスを開くだけで**開講曜日・時限、担当教員、単位数、教室**のデータを自動取得し、Moodleのコースページやダッシュボードからいつでも確認できるようにします。また、取得された教室名からキャンパスマップ（PDF）へのダイレクトアクセスも可能です。
*   **フォーラムの一括既読機能**: フォーラムの画面に「すべて既読にする」ボタンを追加し、未読の投稿をワンクリックでまとめて既読にできます。
*   **先生へのメッセージ（DM）ボタン**: コースページのヘッダーに「DM」ボタンを設置し、クリックするだけでその授業の担当教員へのメッセージ送信画面を新規タブで開きます。
*   **ダークモード**: 目に優しい暗めの配色に変更します。

## インストール方法

### ストアからインストール
以下のリンクからインストールするのが最も簡単で、更新も自動的に行われます。

*   **Chrome**: [Chromeウェブストア](https://chromewebstore.google.com/detail/extension-for-moodle+r/olbkbbpbjjikdplcfcajghjelmoboean)から「Chromeに追加」をクリックします。
*   **Edge**: Microsoft Edge アドオンから「入手」をクリックします。Chromeの拡張機能を使用できる機能を使用してご利用ください。


### 手動インストール
こちらは常に最新版を試したい場合や、ご自身で改造する場合の方法です。不具合が多数含まれる場合があります。

1.  本リポジトリをクローン、またはZIPファイルでダウンロードして展開します。
2.  ブラウザの拡張機能管理ページ（Chromeの場合は `chrome://extensions/`）を開きます。
3.  右上の「デベロッパー モード」をオンにします。
4.  「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードしたフォルダ（`manifest.json` が含まれているフォルダ）を選択します。
